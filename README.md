# PatreonListLib

Thank your patrons!

## Usage:
```
startActivity(Intent(this, PatreonsListActivity::class.java).apply {
    putExtra(
        PatreonsListActivity.DATA_ARG, PatreonListData(
            List(100) {Backer("name")},
            Uri.parse("http://emmanuelmessulam.com.ar/thanks.html")
        )
    )
})
```
